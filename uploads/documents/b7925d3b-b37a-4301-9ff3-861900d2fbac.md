## 课堂内容稿

【开场】同学们好，今天我们学习高等数学中一个非常重要的概念——极限的精确定义。
【导入】首先，我们来回顾一个直观的例子。请描述函数 f(x)=2x+1 当 x 趋近于 1 时的变化趋势？（停顿，等待学生回答）没错，f(x) 的值无限趋近于 3。这里的“无限趋近”是我们的直观描述，但数学需要精确。如何用严格的数学语言来刻画这种“无限趋近”呢？这就引出了我们今天的主角——极限的ε-δ定义。
【新授-定义阐述】我们来看极限 lim(x→a)f(x)=A 的ε-δ定义：对任意给定的ε>0，总存在δ>0，使得当0<|x-a|<δ时，恒有|f(x)-A|<ε成立。这个定义比较抽象，我们来拆解一下。ε刻画了函数值f(x)与极限A的接近程度，δ刻画了自变量x与a的接近程度。核心思想是：只要你想让f(x)无限接近A（ε可以任意小），我总能找到一个x的范围（由δ决定），在这个范围内就能保证这个接近程度。这就是从“无限趋近”到“可以控制”的精确转化。
【新授-举例1】我们通过几个具体例子来理解如何用定义证明极限。第一个例子：证明lim(x→2)5 = 5。这是常数函数。对于任意给定的ε>0，我们需要找到δ>0。看看目标：|f(x)-5|=|5-5|=0，0永远小于任意正数ε，所以无论x怎么变，这个不等式都成立。因此，δ可以取任意正数，比如δ=1。这证明了极限就是它自身。
【新授-举例2】第二个例子是我们导入中的函数：证明lim(x→1)(2x+1) = 3。目标是：对于任意ε>0，找到δ>0。我们进行推导分析：要使|(2x+1)-3| = |2x-2| = 2|x-1| < ε成立，只需要|x-1| < ε/2。所以，我们只需取δ = ε/2（或者更小的正数）。这样，当0<|x-1|<δ时，就一定有|f(x)-3|<ε。这就是完整的证明思路。
【新授-举例3】第三个例子稍复杂：分析证明lim(x→2)(x^2) = 4的思路。目标是使|x^2 - 4| < ε。我们先因式分解：|x^2-4| = |x-2|·|x+2|。这里出现了|x+2|，它不是常数。为了处理它，我们常用的技巧是先限制x在2的附近，比如先假设|x-2|<1（即1<x<3），那么|x+2|就小于5。于是，我们得到|x^2-4| < 5|x-2|。接下来，要使5|x-2| < ε，只需|x-2| < ε/5。最后，为了同时满足“限制范围”和“小于ε”这两个条件，我们取δ为1和ε/5中较小的那个，即δ = min{1, ε/5}。这样，当0<|x-2|<δ时，就能保证|x^2-4| < ε。
【新授-归纳总结】通过这几个例子，我们可以总结出用ε-δ定义证明极限的通用步骤：1. 面对任意给定的ε>0；2. 分析|f(x)-A|，尝试化简或放大；3. 反解出或推导出关于|x-a|的不等式；4. 取定δ（通常是ε的函数，有时需要与一个常数取最小值）；5. 按定义格式完整叙述。核心思想是：ε的任意性保证了“无限接近”，δ的存在性体现了“可以控制”。
【巩固练习】现在请大家进行课堂练习。第一题：根据定义证明lim(x→3)(4x-5) = 7。请大家动笔尝试，注意步骤。第二题：分析证明lim(x→-1)(x^2+2x) = -1的关键步骤，并思考如何选取δ。提示：可以先对f(x)-A进行因式分解。
【小结与作业】我们来总结一下。知识上，极限的ε-δ定义是什么？其几何意义是什么？方法上，证明的一般步骤是什么？处理复杂函数时的常用技巧是什么？（先限制|x-a|的范围）。思想上，我们是如何将直观的“无限趋近”转化为精确的数学语言的？作业：必做题：1. 用ε-δ定义证明 lim(x→c) k = k （k为常数）；2. 证明 lim(x→0) x^3 = 0。选做题：尝试分析并证明函数 f(x)=1/x 在 x→2 时的极限。最后，预习函数极限的性质。

# 三角函数第一课

**文档类型**：lesson  
**学科**：数学  
**年级**：七年级  

## 教学目标

1. 知识目标：理解直角三角形中三角函数的背景与意义；掌握正弦、余弦、正切的基本定义（对边比斜边、邻边比斜边、对边比邻边）。
2. 能力目标：能够根据已知直角三角形的边长，正确计算锐角的正弦、余弦、正切值；初步体会数形结合思想。
3. 情感目标：通过介绍三角函数的历史背景与实际应用（如测量、音乐），激发学生学习数学的兴趣，感受数学的实用性与文化价值。

*时长：3 分钟*

## 导入

1. 情境引入：展示古埃及人利用相似三角形原理测量金字塔高度的故事图片或简短动画，提问：'如果不直接测量，如何知道金字塔有多高？' 引导学生思考直角三角形边角关系的应用。
2. 回顾旧知：快速回顾直角三角形的构成（直角、锐角、对边、邻边、斜边）以及相似三角形的性质（对应边成比例）。
3. 提出问题：'在一个确定的直角三角形中，锐角的大小与其对边、邻边、斜边的长度之间有怎样的定量关系？' 引出本节课主题——三角函数。

*时长：7 分钟*

## 新授

第一部分：三角函数的背景与定义
1. 历史背景简述：简要介绍三角学源于天文学和测量，'三角'指三角形，'函数'指角度与边长比值之间的依赖关系。
2. 定义探究：在黑板或PPT上绘制一个标准的直角三角形ABC（∠C=90°，∠A为锐角）。
   - 明确∠A的对边a、邻边b、斜边c。
   - 定义：
     正弦 sin A = ∠A的对边 / 斜边 = a/c
     余弦 cos A = ∠A的邻边 / 斜边 = b/c
     正切 tan A = ∠A的对边 / ∠A的邻边 = a/b
   - 口诀记忆：'正弦对斜，余弦邻斜，正切对邻'。
3. 理解要点：强调比值只与锐角∠A的大小有关，与三角形大小无关（可通过相似三角形验证）。

第二部分：公式推导与简单计算
1. 推导关系：引导学生从定义推导 sin²A + cos²A = 1 (利用勾股定理 a²+b²=c²)。
2. 计算示例（教师板书示范）：
   例1：在Rt△ABC中，∠C=90°，AC=3，BC=4，AB=5。求∠A和∠B的正弦、余弦、正切值。
   解：对于∠A，对边BC=4，邻边AC=3，斜边AB=5。
       sin A = 4/5, cos A = 3/5, tan A = 4/3。
       同理，对于∠B，对边AC=3，邻边BC=4，斜边AB=5。
       sin B = 3/5, cos B = 4/5, tan B = 3/4。
3. 观察发现：sin A = cos B, cos A = sin B，直观感受互余角三角函数关系（为后续学习铺垫）。

*时长：20 分钟*

## 巩固练习

1. 基础练习（学生独立完成，教师巡视指导）：
   (1) 在Rt△DEF中，∠E=90°，DE=6，EF=8。求∠D的正弦、余弦、正切值。
   (2) 已知在Rt△PQR中，∠R=90°，sin P = 5/13，且PQ=26，求PR和QR的长度。
2. 小组讨论：为什么我们说三角函数值（如sin 30°）是一个确定的数？请结合相似三角形的性质说明。
3. 快速问答（教师提问，学生抢答）：
   - 在直角三角形中，锐角的正弦值可能大于1吗？为什么？
   - tan 45°的值可能是什么？猜一猜并说明理由。

*时长：10 分钟*

## 小结作业

1. 课堂小结：
   - 师生共同回顾：今天我们学习了什么？（三角函数的背景、正弦、余弦、正切的定义）
   - 关键点提炼：三角函数是锐角的函数，表示的是边长之间的比值；比值大小只与角的大小有关。
2. 布置作业：
   - 必做题：课本对应章节的习题，完成3道关于已知边长求三角函数值的计算题。
   - 选做题/探究：查阅资料，了解三角函数在现实生活（如坡度的计算、音乐声波）中的一个应用实例，并简单记录。
   - 预习：思考当角度变化时，这些比值是如何变化的。
3. 结束语：'三角是美丽的，函数是有力的，两者的结合将为我们打开一扇认识世界的新窗户。下节课我们将进一步探索它们的性质。'

*时长：5 分钟*

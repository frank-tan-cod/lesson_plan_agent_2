# 高等数学

**文档类型**：lesson  
**学科**：数学  
**年级**：大一  

## 教学目标

1. 知识与技能：理解极限的ε-δ定义，掌握利用定义证明简单函数极限的方法。
2. 过程与方法：通过多个具体实例的逐步分析，体会从直观描述到严格定义的数学抽象过程，培养逻辑推理与分析能力。
3. 情感态度与价值观：感受数学的严谨性与精确美，克服对形式化定义的畏难情绪，增强学习自信心。

*时长：5 分钟*

## 导入

【情景导入，回顾直观】
1. 提问复习：请用语言描述函数f(x)=2x+1在x趋近于1时的变化趋势。（预期回答：f(x)的值无限趋近于3）
2. 教师设问：这里的“无限趋近”是一个描述性的语言，如何用精确的、量化的数学语言来刻画这种趋势呢？这就像我们描述“两个人非常接近”，需要定义一个标准的“距离”来衡量。今天我们就来学习这个精确定义——极限的ε-δ定义。
【举例铺垫】
展示函数f(x)=2x+1的图像，动态演示x在1附近取值时，f(x)在3附近的变化。

*时长：5 分钟*

## 新授

【第一部分：剖析定义，理解逻辑】
1. 板书并逐句讲解极限lim(x→a)f(x)=A的ε-δ定义：
   “对任意给定的ε>0，总存在δ>0，使得当0<|x-a|<δ时，恒有|f(x)-A|<ε成立。”
2. 结合几何意义解释：ε刻画了函数值f(x)与极限A的接近程度，δ刻画了自变量x与a的接近程度。定义的核心是：只要你想让f(x)无限接近A（ε任意小），我总能找到一个x的范围（由δ决定），在这个范围内就能保证这个接近程度。

【第二部分：多重举例，分步应用】
*例子1（常数函数）：证明lim(x→2)5 = 5。
   - 分析：A=5， f(x)恒等于5。对任意ε>0，要证|f(x)-5|<ε。
   - 验证：|f(x)-5|=|5-5|=0<ε恒成立，与x的取值无关。
   - 结论：可取任意正数作为δ（如δ=1），都能满足条件。教师强调这是最特殊的情形。

*例子2（线性函数）：证明lim(x→1)(2x+1) = 3。 （此为导入中函数）
   - 已知：A=3， a=1。 目标：对于任意给定的ε>0，找到合适的δ>0。
   - 分析推导：要使|f(x)-A|=|(2x+1)-3|=|2x-2|=2|x-1|<ε，只需|x-1|<ε/2。
   - 构造取法：只要取δ=ε/2（或更小的正数），则当0<|x-1|<δ时，必有|f(x)-3|<ε成立。
   - 完整板书证明过程，强调“找δ”的逆向推导思路。

*例子3（二次函数）：分析lim(x→2)(x^2) = 4 的证明思路。
   - 已知：A=4， a=2。目标不等式：|x^2 - 4| < ε。
   - 因式分解：|x^2-4| = |x-2|·|x+2|。
   - 技巧性处理（局部有界性）：我们先限制x在2的附近，比如先限制|x-2|<1（即1<x<3），那么|x+2|<5。于是有|x^2-4| = |x-2|·|x+2| < 5|x-2|。
   - 要使5|x-2|<ε，只需|x-2|<ε/5。
   - 综合两个限制：取δ = min{1, ε/5}。则当0<|x-2|<δ时，能同时满足上述两个条件，从而保证|x^2-4| < ε。
   - 教师总结：对于复杂函数，证明时常需要先对|x-a|加一个限制（如δ≤1），以简化表达式。

【第三部分：要点归纳】
1. 证明的通用步骤：①对任意ε>0；②分析|f(x)-A|，尝试化简或放大；③反解出或推导出关于|x-a|的不等式；④取定δ（通常表示为ε的函数，有时需要与一个常数取最小值）；⑤按定义格式完整叙述。
2. 核心思想：ε的任意性保证了“无限接近”，δ的存在性体现了“可以控制”。

*时长：25 分钟*

## 小结作业

【课堂小结】
1. 知识小结：极限的ε-δ定义是什么？其几何意义是什么？
2. 方法小结：用ε-δ定义证明函数极限的一般步骤是什么？处理复杂函数时的一个常用技巧是什么（先限制|x-a|范围）？
3. 思想小结：本节课是如何通过具体例子，一步步将直观的极限思想转化为精确的数学语言的？
【布置作业】
1. （必做）课后习题：用ε-δ定义证明 lim(x→c) k = k （k为常数），以及 lim(x→0) x^3 = 0。
2. （选做/思考）尝试分析函数 f(x)=1/x 在 x→2 时的极限，并用定义证明你的结论。体会寻找δ的过程。
3. 预习：函数极限的性质。

*时长：3 分钟*

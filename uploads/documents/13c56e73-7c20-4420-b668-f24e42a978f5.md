## 课堂内容稿

【开场】同学们好！今天我们将开启一个数学新篇章——三角函数。我们先来看一个古老的故事：古埃及人是如何测量金字塔高度的呢？他们利用的是直角三角形的边角关系。这其实就是三角函数思想的萌芽。我们知道一个直角三角形有直角、锐角，以及三条边：对边、邻边和斜边。那么，一个锐角的大小，和它对应的边长的比值之间，是否存在某种固定的关系呢？这就是我们今天要探究的核心问题。【新授第一部分：背景与定义】首先，了解一下背景。三角学源于天文学和测量。‘三角’指的是三角形，‘函数’指的就是角度与边长比值之间的依赖关系。现在，我们来看定义。以直角三角形ABC为例，∠C=90°，∠A是我们要研究的锐角。∠A的对边是a（边BC），邻边是b（边AC），斜边是c（边AB）。我们定义：∠A的正弦（sin A）等于它的对边比斜边，即a/c。∠A的余弦（cos A）等于它的邻边比斜边，即b/c。∠A的正切（tan A）等于它的对边比邻边，即a/b。为了便于记忆，我们可以用口诀：‘正弦对斜，余弦邻斜，正切对邻’。这里有一个非常重要的理解要点：这些比值只与锐角∠A的大小有关，而与我们画的这个直角三角形的大小无关。因为所有相似的直角三角形，对应边的比值是相同的。我们可以用相似三角形的性质来验证这一点。【新授第二部分：公式推导与简单计算】接下来，我们做一些推导和计算。从定义出发，我们可以推导出一个重要的恒等式：sin²A + cos²A = 1。推导过程很简单，将sin A = a/c， cos A = b/c代入，得到 (a²/c²) + (b²/c²) = (a²+b²)/c²。根据勾股定理 a²+b²=c²，所以结果等于1。好，现在我们来做一道例题。已知Rt△ABC中，∠C=90°，AC=3，BC=4，AB=5。求∠A和∠B的正弦、余弦、正切值。对于∠A：对边BC=4，邻边AC=3，斜边AB=5。所以，sin A = 4/5，cos A = 3/5，tan A = 4/3。对于∠B：对边是AC=3，邻边是BC=4，斜边仍是AB=5。所以，sin B = 3/5，cos B = 4/5，tan B = 3/4。大家观察一下结果，有什么发现？sin A = cos B，cos A = sin B。这是因为∠A和∠B是互余的两个角（和为90°）。这为我们后续学习互余角的三角函数关系埋下了一个伏笔。【巩固练习】现在请大家完成两个基础练习题。1. 在Rt△DEF中，∠E=90°，DE=6，EF=8。求∠D的正弦、余弦、正切值。2. 已知在Rt△PQR中，∠R=90°，sin P = 5/13，且斜边PQ=26，求直角边PR和QR的长度。完成后，我们进行小组讨论：为什么我们说三角函数值（比如sin 30°）是一个确定的数？请大家结合相似三角形的性质来说明。接下来，我们进入快速抢答环节：第一个问题：在直角三角形中，锐角的正弦值可能大于1吗？为什么？第二个问题：tan 45°的值可能是什么？猜一猜并说明理由。【小结与作业】好了，课程接近尾声。让我们一起来回顾一下今天学到了什么。我们学习了三角函数的背景，以及正弦、余弦、正切这三个基本定义。关键点在于：三角函数是锐角的函数，它表示的是边长之间的比值，而这个比值的大小只与角的大小有关，与三角形的大小无关。今天的作业是：必做题：完成课本对应章节的3道关于已知边长求三角函数值的计算题。选做题/探究：查阅资料，了解三角函数在现实生活（比如坡度的计算、音乐声波）中的一个应用实例，并简单记录下来。预习：思考当角度变化时，正弦、余弦、正切这些比值会如何变化。最后，送给大家一句话：三角是美丽的，函数是有力的，两者的结合将为我们打开一扇认识世界的新窗户。下节课我们将进一步探索它们的性质。
